# AI Career Agent | Matt Lorensen

An interactive, RAG-enabled professional portfolio agent designed to bridge the gap between a static resume and an immersive career exploration experience. This tool allows recruiters and hiring managers to query my professional history, leadership philosophy, and technical expertise in real-time.

## 🎯 Purpose
I built this agent to demonstrate my ability to architect and deploy AI-driven solutions. It focuses on showcasing my expertise in:
* **IT Operations & Frameworks:** Designing scalable service delivery models.
* **Incident Response:** My approach to triage, stabilization, and governance.
* **Leadership:** Managing cross-functional teams and aligning technical delivery with business objectives.
* **Strategic Communication:** Translating complex technical requirements for C-Suite and stakeholder alignment.

## 🛠 Tech Stack
* **Framework:** Python, Streamlit
* **AI Engine:** Google Gemini 3.5 Flash
* **Orchestration:** LangChain
* **Vector Store:** ChromaDB
* **Architecture:** Retrieval-Augmented Generation (RAG)
